Venda Segura

Política de privacidade

O Venda Segura foi criado para organizar dados de clientes, parcelas, vencimentos e histórico de cobrança.

Armazenamento

Nesta versão, os dados são salvos localmente no navegador usado para acessar o sistema.

Backup

O usuário pode exportar e importar backups em arquivo JSON. O arquivo exportado deve ser guardado em local seguro.

Compartilhamento

O sistema não vende nem compartilha dados automaticamente com terceiros.

WhatsApp

Ao clicar em WhatsApp, o usuário é direcionado para o aplicativo com uma mensagem pronta. O envio final depende da confirmação do próprio usuário.
