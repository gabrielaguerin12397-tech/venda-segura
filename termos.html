Venda Segura

Termos de uso

O Venda Segura é um sistema de organização de clientes, parcelas e cobranças para vendedores e pequenos negócios.

Responsabilidade do usuário

O usuário é responsável pelas informações cadastradas, pela guarda dos backups e pelo uso adequado das mensagens de cobrança.

Cobranças

O sistema auxilia na organização das cobranças, mas não realiza cobrança automática, negativação ou consulta pública de devedores.

Dados

Os dados ficam armazenados no navegador do usuário. Para evitar perda de informação, o usuário deve exportar backups regularmente.

Uso permitido

O sistema deve ser usado apenas para controle próprio de vendas, clientes e parcelas, respeitando a legislação aplicável.
